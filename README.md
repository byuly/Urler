## ðŸ“¦ Getting Started

### Prerequisites

* Node.js (>=18)
* Java (>=17)
* Docker (optional, for containerized deployment)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/urler.git
   cd urler
   ```

2. **Frontend Setup**

   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Backend Setup**

   ```bash
   cd backend
   ./mvnw spring-boot:run
   ```

4. **(Optional) Run with Docker**

   ```bash
   docker-compose up --build
   ```
